# 流向 v0.1

不是管理时间，是看见时间去了哪里。

## 使用方式

1. 解压压缩包。
2. 双击 `index.html` 打开。
3. 首页点「开始记录」即可开始，不需要先分类。
4. 停止时只做五选一：主线、生活、恢复、迷路、先不管。
5. 「快记一句」用于记录饮食、情绪、身体、打断、灵感等小事件。
6. 「补一段」用于事后补录。
7. 「时间轴」用于晚上回看。
8. 「更多」里可以导出/导入 JSON。

## 重要提醒

数据保存在当前浏览器本地。清理浏览器数据、换浏览器、换设备前，请先导出 JSON。

## v0.1 的设计原则

- 先开始，不先分类。
- 先说人话，不先填表。
- 停止时只粗分。
- 细节全部可选。
